## Run on Android ##

1) ```npm install```
2) ```npm run android```

## Run on iOS ##

1) ```npm install```
2) ```cd ios```
3) ```rm -rf ~/Library/Caches/CocoaPods Pods ~/Library/Developer/Xcode/DerivedData/*; pod deintegrate; pod setup; pod install;```
4) ```open in XCode filename extension with .xcworkspace in folder ios```
